# Smart Lender

## Project Description

Smart Lender is a machine learning-based web application that predicts loan approval status using applicant information. The project helps banks and financial institutions automate loan approval decisions using predictive analytics.

## Features

- Loan approval prediction
- Data preprocessing
- Missing value handling
- Feature encoding
- Dataset balancing
- Feature scaling
- Model comparison
- Flask web application

## Algorithms Used

- Decision Tree
- Random Forest
- K-Nearest Neighbors
- XGBoost

## Model Accuracy

| Model | Accuracy |
|-------|----------|
| Decision Tree | 82.84% |
| Random Forest | 86.98% |
| KNN | 69.23% |
| XGBoost | 84.02% |

## Best Model

Random Forest was selected as the final model because it achieved the highest testing accuracy of 86.98%.
